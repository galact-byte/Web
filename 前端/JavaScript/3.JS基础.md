## 1.常量

使用 const 声明的变量称为常量，常量不允许重新赋值，声明的时候必须初始化。

```
const 变量名=值
```

## 2.数据类型

1. JS 分为基本数据类型和引用(复杂)数据类型。

- 基本数据类型又分为：

  - number数字型：小数、整数、正数、负数

    > 常和算术运算符一起，比如：`+`求和、`-`求差、`*`求积、`/`求商、`%`取模(取余数)
    > **NaN 代表一个计算错误， 它是一个不正确或者未定义的数学操作得出的结果，任何对 NaN 的操作都会返回 NaN。**

    - string 字符串型：通过单引号、双引号或反引号包裹数据都叫 字符串。单引号和双引号可以互相嵌套，但是不能自己嵌套自己。
      > **必要的时候可以使用转义符`\`输出单引号和双引号 。`+` 运算符可以实现字符串的拼接。**
          模板字符串用来拼接字符串和变量，外面用` `` `，里面`${}`变量名。

  - boolean 布尔型：表示肯定或否定时在计算机中对应的数据。
  - underfined 未定义型：只声明变量，不赋值的情况下，变量默认值为 undefined。
  - null 空类型：若一个变量莉存放的是对象，没准备好对象可以放个 null。
    > **null 表示赋值了，但是内容为空(相当于 0)，undefined 表示没有赋值(相当于无穷)。**

> JS 是弱数据类型，变量属于什么类型只有赋值后才能确定，Java 是强数据类型。

- 引用数据类型只有 object 对象

2. 检测数据类型
   通过`typeof`检测，返回被检测的数据类型。语法有：

- 运算符：typeof x
- 函数形式：typeof(x)

## 3.类型转换

由于 JS 是弱数据类型，使用表单、prompt 获取过来的数据默认是字符串类型的，不能直接简单的进行加法运算。

1.隐式转换

某些运算符被执行时，系统内部自动将数据类型进行转换，这种转换称为隐式转换。

- `+`号两边只要有一个是字符串就会把另一个也变为字符串。
- `-`、`*`、`/`等都会把数据转成数字类型。
- `+`号作为正号解析可以转换成数字型。
- 任何数据和字符串相加结果都是字符串。

  2.显式转换

自己写代码告诉系统该转成什么类型

- Number：转成数字类型，若字符串内容里有非数字，转换失败时结果为 NaN 即不是一个数字，NaN 也是 number 类型的数据，代表非数字。
- parseInt：只保留整数
- parseFloat：可以保留小数

- `h2`是二级标题
- `table`用于定义 html 表格
