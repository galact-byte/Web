## 1.概念

1.1 JS是一种运行在客户端（浏览器）的编程语言，实现人机交互效果。HTML和CSS是标记语言。 

1.2 JS可以进行
	- 网页特效（监听用户的一些行为让网页做出对应的反馈）例如：下拉菜单、轮播图
	- 表单验证（针对表单数据的合法性进行判断）
	- 数据交互（获取后台的数据，渲染到前端）
	- 服务端编程（node.js）

1.3 JS由两大类组成，分别为ECMAScript和Web APIs。
- ECMAScript是Javascript语言基础规定了js基础语法核心知识，比如变量、分支语句、循环语句、对象等等。
- Web APIs又分为DOM和BOM。
	- DOM是页面文档对象模型，操作文档，比如对页面元素进行移动、大小、添加删除等操作。
	    比如论坛在文本框里发送消息时多一条消息用的就是DOM操作。
	- BOM是浏览器对象模型，操作浏览器，比如页面弹窗，检测窗口宽度、存储数据到浏览器等等。
		比如小窗确实是否关闭、删除之类的。 

## 2.书写位置

2.1 JS书写位置：
- 内部JavaScript，直接写在html文件里，用script标签包住（写在`</body>`上面）。
	我们将`<script>`放在HTML文件底部附近的原因是浏览器会按照代码在文件中顺序加载HTML。
		在JavaScript中，`alert`是一个内置的函数，用于显示包含文本消息和一个确定按钮的对话框框。这个对话框框是浏览器提供的一种简单的弹出式窗口，通常用于向用户显示一些信息或请求用户执行某些操作。
	![[Web/前端/图片/Pasted image 20231118184213.png]]
- 外部JavaScript，代码写在以`.js`结尾的文件里，通过script标签，引入到html页面中。
		外部和内部的不一样，外部的`script`标签里不能写东西，会被忽略。
	![[Web/前端/图片/Pasted image 20231118185840.png]]
- 行内(内联)JavaScript，代码写在标签内部。

	JS书写位置和css类似， css位置有: 行内样式表，直接写在标签里。内部样式表，写在head之间。 外部样式表，link的方式引进。

## 3.注释

单行注释用`//`，快捷键`ctrl+/`，块注释用`/* */`，快捷键`shift+alt+a`。JS中结束符用`;`，但在实际开发中可写可不写浏览器可以自动推断语句结束位置。

## 4.输入输出

4.1 输出语法

语法1:
```
document.write("要出的内容")
```
向body输出内容，若输出内容写的是标签，也会被解析成网页元素。通过js方式在html里输出内容。
	`<h1>`标签用来定义页面主标题的元素，html中标题标签包括`<h1>`到`<h6>`，通常一个页面应该只有一个`<h1>`标签。

语法2：
```
alert("要出的内容")  
```
页面弹出警告对话框。

语法3：
```
console.log("控制台打印")
```
控制台输出语法，程序员调试使用。

4.2 输入语法
```
prompt("请输入您的姓名:")
```
显示一个对话框，对话框中包含一条文字信息，用来提示用户输入文字。

	JavaScript代码执行顺序，按HTML文档流顺序执行代码。
	alert()和prompt()它们会跳过页面渲染先被执行。
