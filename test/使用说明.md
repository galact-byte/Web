# 漫画上色工作流使用说明

## 快速开始

### 1. 文件准备

创建以下文件夹结构（路径可自定义）：

```
D:/manga_colorization/
├── input/          ← 放入黑白漫画页面
└── reference/      ← 放入参考彩图
```

### 2. 模型文件位置

确保以下模型已放在 ComfyUI 对应目录：

| 模型 | 路径 |
|------|------|
| `svdq-int4_r128-qwen-image.safetensors` | `ComfyUI/models/diffusion_models/` |
| `qwen_2.5_vl_7b_fp8_scaled.safetensors` | `ComfyUI/models/clip/` |
| `qwen_image_vae.safetensors` | `ComfyUI/models/vae/` |
| `PanelPainter_v3_Qwen2511.safetensors` | `ComfyUI/models/loras/` |

### 3. 导入工作流

将 `manga_colorization_local_batch.json` 拖入 ComfyUI 界面

---

## 使用步骤

### Step 1: 设置输入路径

找到 **"批量加载漫画页面"** 节点，修改路径为你的黑白漫画所在文件夹：
```
D:/manga_colorization/input/
```

### Step 2: 上传参考彩图

找到 **"参考彩图 (菲伦配色)"** 节点，上传你的菲伦彩色参考图

### Step 3: 修改提示词（可选）

找到 **"正向提示词"** 节点，当前默认：
```
color this manga panel in anime style, preserving the original line art and text. 
Fern has purple/violet hair and pale skin, wearing dark mage robes.
```

### Step 4: 运行

点击 **Queue Prompt** 开始批量处理

---

## 参数调优

如果效果不理想，可以调整 **K采样器** 节点的参数：

| 参数 | 当前值 | 建议范围 | 说明 |
|------|--------|----------|------|
| Steps | 5 | 4-8 | 步数越多越细腻，但更慢 |
| CFG | 1.2 | 1.0-1.5 | 提示词影响强度 |
| Denoise | 0.88 | 0.75-0.95 | 重绘幅度，越高变化越大 |
| LoRA强度 | 0.55 | 0.4-0.7 | 在 LoRA 节点调整 |

---

## 常见问题

**Q: 显存不足**  
A: 将 Steps 降到 4，或每次只处理少量图片

**Q: 颜色不对**  
A: 检查参考图是否正确加载，或调整提示词描述

**Q: 线稿被覆盖**  
A: 降低 Denoise 值到 0.75-0.80
