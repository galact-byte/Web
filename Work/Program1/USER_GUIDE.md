# 定级备案管理系统 用户操作手册

本手册用于测试人员和业务人员快速上手，按步骤执行即可完成一轮完整业务流程。

## 1. 快速开始（3分钟）

1. 启动系统：双击项目根目录 `start.bat`。  
2. 打开地址：`http://127.0.0.1:8011`。  
3. 点击右上角“登录”，使用账号登录。  

默认内置账号（如未被管理员修改）：

- 管理员：`admin / admin123`
- 审核人：`leader / leader123`
- 测评师：`tester / tester123`

## 2. 你登录后该做什么

### 2.1 测评师（tester）日常流程

1. 进入“单位信息”，新建单位。  
2. 进入“系统信息”，新建系统并关联单位。  
3. 进入“报告与审核”，按系统生成报告（备案表/定级报告/专家评审意见表）。  
4. 在报告页补充章节、设置签章信息、调整内容。  
5. 点击提交审核，交给审核人。  

### 2.2 审核人（leader）日常流程

1. 登录后进入“报告与审核”。  
2. 选择待审核报告，执行“通过”或“驳回”。  
3. 若驳回，填写明确修改意见。  
4. 可在“流程管控”查看当前节点和进度。  

### 2.3 管理员（admin）日常流程

1. 登录后先检查“模板管理”中的默认模板。  
2. 如需更新模板，在“模板管理”上传新模板或导入本地官方模板。  
3. 在“流程管控”维护节点负责人、时限规则。  
4. 在“登录与用户”维护账号和角色。  
5. 审核单位/系统删除申请，管理回收站清理。  

## 3. 标准业务闭环（建议演示路径）

1. `tester` 创建单位与系统。  
2. `tester` 生成报告并提交审核。  
3. `leader` 驳回一次并给出意见。  
4. `tester` 修改后再次提交。  
5. `leader` 审核通过。  
6. 导出报告 Word/PDF 进行交付。  

## 4. 模板相关操作

### 4.1 导入官方模板

1. 将模板放在项目根目录，文件名建议：
- `01-*.docx`（备案表）
- `02-*.docx`（定级报告）
- `03-*.docx`（专家评审意见表）
2. 管理员进入“模板管理”，点击“导入本地官方模板”。

### 4.2 模板生效规则

1. 报告生成时优先使用匹配模板。  
2. 若模板不可用，系统会回退到通用导出逻辑。  

## 5. 看板与知识库

### 5.1 数据看板

1. 支持按地区、行业、级别、测评师、状态筛选。  
2. 支持下钻查看明细。  
3. 支持导出 Excel / PDF / PNG 截图。  

### 5.2 知识库

1. 管理员上传文档并设置分类、地市、级别。  
2. 支持模糊检索和精确检索。  
3. 支持版本管理、回滚、下载留痕。  

## 6. 常见问题（测试现场高频）

### 6.1 登录后提示无权限

- 原因：当前账号角色不具备该功能。  
- 处理：切换账号或请管理员分配权限。  

### 6.2 导出/下载失败

- 原因：未登录、会话过期、或模板文件缺失。  
- 处理：重新登录后重试；管理员检查模板配置。  

### 6.3 普通账号看不到某些菜单

- 原因：角色可见范围控制。  
- 说明：这是正常权限设计，不是页面异常。  

### 6.4 客户采集链接打不开或提交失败

- 原因：链接过期或已停用。  
- 处理：测评师重新生成采集链接。  

## 7. 测试前检查清单

1. 服务是否已启动（`http://127.0.0.1:8011` 可访问）。  
2. 是否已登录正确角色账号。  
3. 模板是否已导入并设置默认。  
4. 测试数据是否已按“单位 -> 系统 -> 报告”顺序准备。  

## 8. 交付前检查清单

1. 报告审核状态为通过。  
2. Word/PDF 导出正常。  
3. 流程节点和操作记录可追溯。  
4. 看板统计与业务数据一致。
