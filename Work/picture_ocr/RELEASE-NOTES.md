## ✨ 手机局域网采集更连续

- 电脑端新增、删除或调整分类、资产和检查项后，已扫码的手机页面会自动同步最新采集结构，无需重新扫码。
- 关闭二维码窗口只会收起窗口，不会停止采集；工作台工具栏会显示“采集中”，可随时重新查看二维码、链接或停止会话。
- 手机每约 2 秒同步一次项目结构；已删除的资产或检查项不会再接受上传，避免图片写入错误目标。

## 📷 图片来源与网页相机

- 手机默认使用“系统选择（推荐）”，适合由浏览器或系统提供拍照、相册或文件选择。
- 可切换为“拍照 / 相册分开选择”，浏览器支持时会优先打开网页后置相机预览；也可分别使用系统相机或相册回退。
- PC Web 在项目工作台刷新后会恢复当前项目，不再返回项目列表。

## 🛠️ 界面与稳定性优化

- 优化不同窗口宽度及 Windows DPI 缩放下的项目列表：窄窗口自动收纳低频操作，宽屏保留完整快捷操作，表头与数据列保持对齐。
- 项目列表不再提供局域网采集入口，避免会话绑定目标不清晰；请进入目标项目后从工作台启动采集。

## 🔄 兼容性与使用提示

- 图片支持 PNG、JPEG、GIF、WebP、BMP，单张最大 10MB；HEIC 暂不支持。
- 网页相机是否可用取决于浏览器的安全策略和权限。vivo 自带浏览器可能仍只提供相册；可切换“系统选择”，或使用微信、Chrome 等支持拍照的浏览器打开链接。
- 手机局域网采集仅适用于可信的同一 Wi-Fi 或个人热点。该临时 HTTP 会话不加密，请勿在公共 Wi-Fi、陌生热点或其他不可信网络使用。

## 📦 下载与使用

- **Web 静态版**：下载 `picture-ocr-v0.4.3.zip`，解压后双击“启动测评证据采集工具.bat”。运行期间请保持 PowerShell 窗口开启。
- **Windows 客户端**：下载本 Release 附带的 `.exe` 便携版后双击运行。
- 在目标项目中点击“手机局域网采集”，选择手机实际可访问的可信 Wi-Fi / 热点地址后扫码。
